Stone-Paper-Scissor Game is a fun Python game where you can play against the computer. The computer picks its moves randomly, and the game keeps track of how many times you win, lose, or draw. To play, you just run the game, type "play" to start, and then choose stone, paper, or scissor. The scores are updated automatically after each round. This game is a simple way to practice Python and have some fun at the same time. It can be played on any computer that has Python installed.

## Features
- Play against a computer with random moves
- Tracks wins, losses, and draws
- Fun interactive prompts with emojis
